DEFINICIONES :

Lenguaje de marca : Un lenguaje de marca es un modo de codificar un documento donde , junto con el
texto se incorporan etiquetas con información adicional relativa a la estructura del texto adicional
relativa a la estructura del texto o su formato de presentación

Un sistema ERP "sistema de planificación de recursos empresariales " se hace cargo de las distintas operaciones
internas de una empresa , desde producción a distribución o incluso recursos humanos

CARACTERÍSTICAS DE HTML, CSS, XML, MARKDOW, XPATH Y XQUERY

HTML:
-puede ser creado y editado con cualquier editor básico de textos.
-Es multiplataforma ,es decir , puede ser visualizado por cualquier navegador de cualquier sistema operativo,

CSS :
Son un lenguaje que organiza los documentos estructurados en hojas de estilo.Sirven tanto para interpretar un
documento en pantalla, un documento por voz o en dispositivos táctiles basados en Braille.
- Es muy útil para los desarrolladores de aplicaciones multiplataforma o en el desarrollo de aplicaciones web.

XML :

Es un lenguaje de marcado sencillo similar al HTML . Su objetivo es facilitar la representación ,almacena -
miento y transmisión de información varia por parte de aplicaciones informáticas , computadoras , etc.
- XML se almacena en formato texto (no binario)lo cual hace que los documentos sean directamente entedibles
- XML es un estándar abierto no sujeto a ningún tipo de licencia .
- Permite la creación de etiquetas propias y permite asignar atributos a las etiquetas .

MARKDOWN:
Es una herramienta de conversión de texto plano a HTML

XPATH:
Es un lenguaje de descripción de rutas para documentos XML desarrollados por el W3C . Este lenguaje
pone a disposición de los usuarios una sintaxis no basada en XML que permite encontrar elementos
concretos en un documento XML
- Viene integrado en un lenguaje host que permite el procesamiento de los elementos XML.

XQUERY :
Es un lenguaje de consulta que permite extraer información de bases de datos o documentos XML.
XQuery está basado en el lenguaje XPath para el acceso a los nodos XML , pudiendo utilizar todos sus
operadores y funciones .
- El lenguaje XQuery es muy amplio y complejo
